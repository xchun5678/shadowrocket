## 📖 使用说明

1. 将 `.sgmodule` 文件下载至本地。
2. 打开 Shadowrocket，进入「模块」页面。
3. 点击右上角「+」，从本地导入，选择下载的模块。
4. 开启模块开关，即可生效。
